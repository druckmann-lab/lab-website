# Logo Files

Place the following logo files in this directory:

## Required Logos

1. **stanford-medicine.svg** - Stanford Medicine logo
   - Source: https://med.stanford.edu/content/dam/sm/identity/images/stanford-medicine/looks/logos/SM_Web_vert_LG.svg
   - Used in: Index page affiliations section

2. **biox.png** - Stanford Bio-X logo  
   - Source: https://biox.stanford.edu/modules/custom/biox/images/biox-logo.png
   - Used in: Index page affiliations section

3. **wu-tsai.svg** - Wu Tsai Neurosciences Institute logo
   - Source: https://neuroscience.stanford.edu/themes/custom/stanford_wutsai/logo.svg
   - Used in: Index page affiliations section

4. **stanford.png** - Stanford University logo
   - Used in: Footer across all pages

## How to Add

1. Download each logo from the source URL
2. Rename to match the filename above
3. Place in this directory (public/logos/)

The website will display placeholder/broken images until these files are added.
